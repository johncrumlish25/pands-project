# Programming and Scripting Project: Iris Dataset Analysis

## Overview
This project analyses the Iris dataset using Python. The aim is to explore the dataset, generate visualisations, and identify patterns between different flower species.

## About the Dataset
The Iris dataset was introduced by Ronald Fisher and is commonly used in statistics and machine learning. 

It contains 150 samples of iris flowers from three different species:
- Setosa  
- Versicolor  
- Virginica 

Each sample includes four measurements:
- Sepal length  
- Sepal width  
- Petal length  
- Petal width  

## Project Structure
This repository contains the following files:

- analysis.ipynb – Jupyter Notebook containing all analysis and visualisations  
- iris.csv – The dataset used in the project  
- summary.txt – Summary statistics generated from the dataset  
- images – Folder containing all generated plots  

## What the Project Does
The notebook performs the following tasks:
- Loads and prepares the dataset  
- Generates summary statistics for each variable  
- Creates histograms to show distributions  
- Produces scatter plots to examine relationships between variables  
- Provides additional analysis and observations
